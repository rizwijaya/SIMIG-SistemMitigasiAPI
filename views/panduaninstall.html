  • Panduan Instalasi

Langkah awal sebelum menggunakan project adalah melakukan instalasi yaitu sebagai berikut.

  1. 1. Download dan install Golang.
           - Download Golang
           - Documentasi Install Golang
  2. 2. Buka folder $GOPATH /src/github.com/
  3. 3. Pada folder tersebut dapatkan source code dari repository dengan mengetikkan perintah berikut pada terminal.
          git clone https://github.com/2021-IT-Pemrograman-Integratif/project-2-rizwijaya.git
        Atau bisa menggunakan perintah berikut.
          go get -u https://github.com/2021-IT-Pemrograman-Integratif/project-2-rizwijaya

Download dependensi package yang digunakan dengan menggunakan depedency manager.
      dep ensure
Apabila tidak memiliki depedency manager bisa menggunakan perintah dari golang.

  1. 1. Lacak dependensi pada project.
        go mod init
  2. 2. Selanjutnya download dependensi dengan perintah berikut.
        go mod tidy
  3. 3. Build project menggunakan perintah berikut.
        go build

  1. 1. Langkah pertama dalam konfigurasi database adalah buat database baru.
  2. 2. Selanjutnya import database dari project, pada project ini database terletak di folder database/mitigasi.sql.
  3. 3. Konfigurasikan database pada website, buka file config.json yang terletak pada folder config.
  4. 4. Pada file config.json lakukan konfigurasi sesuai dengan database dan host yang digunakan yaitu sebagai berikut. 
    5. 
{
    "DB_USERNAME" : "Username database yang digunakan",
    "DB_PASSWORD" : "Password database yang digunakan",
    "DB_PORT" : "Port dari database",
    "DB_HOST" : "Host dari database",
    "DB_NAME" : "Nama dari database"
}
  5. 5. Simpan Konfigurasi tersebut.

Setelah selesai melakukan instalasi dan konfigurasi file serta database selanjutnya jalankan web server pada folder project menggunakan perintah.
go run main.go
Buka browser dan masukkan alamat sebagai berikut.
https://localhost:8080/
